Que es bootstrap 4?

Es un conjunto de herramientas que se integran en un código abierto el cual puede ser modificable, contiene diversas plantillas que traen consigo botones, formularios, cuadros y menús de navegación basados en HTML y CSS, esta herramienta permite tener una mejor visibilidad y accesibilidad en nuestro sitio web. Uno de los cambios más importantes que este trae es el preprocesador en la versión 3, anteriormente usaba LEES pero esta nueva versión, apropósito las más llamativa, se caracteriza por usar SASS ya que compila más deprisa, Bootstrap 4 no tiene soporte para internet explore 8.

Novedades

1. CAMBIOS EN EL GRID: Se ha mejorado mucho el sistema GRID que utiliza Bootstrap. No ha cambiado la sintaxis, pero ahora se utiliza las unidades REM, en vez de los pixeles, con lo que se consigue una mejor experiencia en dispositivos móviles.

2. GLYPHICONS ES SUSTITUIDO POR FONT AWESOME: La versión 4 dice adiós a los iconos de Glyphicons, dejando paso a Font Awesome, un acierto, ya que la mayoría de los diseñadores utilizaba Font Awesome, mucho más completo que el set de iconos de Glyphicons.

4. ADIOS A LESS: Sass le ha ganado la batalla a Less, llegó el último y lo hizo para quedarse. En esta nueva versión desaparece Less, ya no convivirán juntos. Para algunos una buena decisión y para los que utilizan como pre-procesador Less una malísima noticia. El mayor culpable de esta decisión ha sido Libsass, que hace que compile muy rápido Sass.

4. NO MÁS SOPORTE PARA INTERNET EXPLORER 8: Por fin el fin del soporte para IE 8, en esta nueva versión se ha decido dejar de dar soporte a esta versión del navegador. Aunque hace más de 5 años que salió la versión 9 de Internet Explorer, se le daba soporte porque es el navegador que viene instalado en Windows 7. Así que adiós a esos malditos hacks.

5. FLEXBOX: Si algo es complicado en Bootstrap es el posicionamiento CSS , ahora con Flexbox esto se acabó, vendrá como manera opcional si quieres utilizarlo, para activarlo tendrás que modificar el _variables.css y buscar la variable de Flexbox y cambiarla a verdadero. Flexbox nos facilitará igualar alturas, alineaciones... Sin duda una gran ayuda para diseñar más rápido.

¿Cómo se usa bootstrap 4?

1. DESCARGA LOS INSTALADORES: Para utilizar Bootstrap, primero debes descargar Ruby que permite vincular códigos entre si por medio de comandos, luego descargar Visual Studio Code que es un lector de código muy parecido a Sublime, por último descarga Sass con un comando en Ruby,que permite compilar de forma rápida.

2. GUARDAR LAS CARPETAS EN XAMPP: Luego de haber descargado los intaladores necesarios y haberlos ejecutado, se deben crear las carpetas que almacenan los códigos dentro de Xampp/htdocs.

3. VISUAL STUDIO CODE: Se ejecuta y se colocan los códigos de Sass y Css.

4.ENLACE: se coloca el enlace que conecta a Sass con Css.

5. RESULTADO: se ejecuta el código en el navegador.
